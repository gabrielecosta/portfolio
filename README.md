# Personal website
Free template offered by minimal template by GitHub.
Structure of the website:
- index: home page
- work experiences: detailed description about work experiences
- education and training: curriculum studiorium
- Education Activities and Projects
- Scientific publications

## Running the Site Locally

1. **Download the repository** and navigate into it:
```bash
git clone <repository-url>
cd <repository-folder>
```

2. Install dependecies:
```bash
bundle install
```

3. Run the site locally:
```bash
bundle exec jekyll serve
```

4. Open your browser at http://localhost:4000
 to view the site.
